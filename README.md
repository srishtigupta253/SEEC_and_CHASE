# SEEC and CHASE: An emotion-cause pair-oriented approach and conversational dataset with heterogeneous emotions for empathetic response generation
Existing datasets do not consider the fluctuating emotions in human-human interactions, and hence the dialog agent ends up with generic replies. In this work, we introduce a conversational dataset, **CHASE**, consisting of 4.1k conversations with heterogeneous emotions that show a change in emotion in the last utterance spoken by users. In this dataset, we address the change in emotions by generating golden empathetic responses by utilizing the attributes of empathy introduced by Dr. Brene Brown. Furthermore, we also propose an architecture that employs an ECPE-based approach, **SEEC**, to cause extraction for including multiple sources and causes of an emotion, enhancing the empathetic response generated by the dialog agent.
